## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/vanshika2723/Codomax-day1.git
cd fullstack-day1
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Server

```bash
node index.js
```

### 4. Open in Browser

Visit:

```
http://localhost:3000
```

