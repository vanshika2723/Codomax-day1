# Day 1 – Environment Setup

## Project Overview

This project is created as part of the Full Stack Development training Day 1 tasks. The objective is to set up the development environment, create a basic Express.js application, and run a simple "Hello World" server.

## Technologies Used

* Node.js
* Express.js
* Git
* GitHub
* VS Code
* Postman

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/vanshika2723/Codomax-day1.git
cd fullstack-day1
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run the Server

```bash
node index.js
```

### 4. Open in Browser

Visit:

```
http://localhost:3000
```

You should see:

```
Hello World!
```
